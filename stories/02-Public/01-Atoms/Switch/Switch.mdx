# Switch

Componente de switch toggle acessível baseado em Radix UI. Para estados binários
on/off com mudança de efeito imediato, ideal para configurações e preferências.

## Quando Usar

✅ **Para toggle on/off** - Estados binários com efeito imediato ✅ **Para
configurações** - Ativar/desativar funcionalidades ✅ **Para preferências** -
Opções que mudam o comportamento ✅ **Para controles de visibilidade** -
Mostrar/ocultar elementos

## Quando NÃO Usar

❌ **Para seleção múltipla** - Use Checkbox ao invés ❌ **Para seleção única
entre opções** - Use Radio Group ❌ **Para ações que precisam confirmação** -
Use Button com Dialog

## Acessibilidade

- ✅ **SEMPRE use com Label** - Associe com htmlFor e id
- ✅ **Navegação por teclado** - Space para toggle
- ✅ **Estados visíveis** - Animação clara de transição
- ✅ **Disabled acessível** - Removido de tab order

## Exemplos de Código

### Básico

```tsx
import { Switch } from '@/components/ui/switch'
import { Label } from '@/components/ui/label'
;<div className="flex items-center space-x-2">
  <Switch id="notifications" />
  <Label htmlFor="notifications">Notificações</Label>
</div>
```

### Com React Hook Form

```tsx
import { useForm, Controller } from 'react-hook-form'

function Settings() {
  const { control } = useForm()

  return (
    <Controller
      name="notifications"
      control={control}
      render={({ field }) => (
        <div className="flex items-center space-x-2">
          <Switch
            id="notifications"
            checked={field.value}
            onCheckedChange={field.onChange}
          />
          <Label htmlFor="notifications">Notificações</Label>
        </div>
      )}
    />
  )
}
```
